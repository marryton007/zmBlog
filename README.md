#zmBlog
-------------------

**1. describe**

tiny blog application build with nodejs + express + passport

**2. install**

```javascript
npm install
```

**3. run**

```language
npm start
```

**4. browser**

open a browser tab and visit follow url:
```language
http://localhost:3000
```
